# Xcode Scheme 位置详细说明

## 📍 Scheme 下拉菜单的位置

在 Xcode 顶部工具栏中，Scheme 下拉菜单位于以下位置：

### 位置描述

1. **顶部工具栏区域**
   - 在 Xcode 窗口的最顶部
   - 工具栏从左到右通常包含：
     - **停止按钮**（⏹️，灰色）
     - **运行按钮**（▶️，绿色）
     - **Scheme 下拉菜单** ← **这里！**
     - **目标设备选择器**（iPhone 15 Pro 等）
     - 其他工具栏按钮

2. **Scheme 下拉菜单的外观**
   - 显示当前选择的 Scheme 名称（通常是 "App"）
   - 右侧有一个下拉箭头（▼）
   - 点击后会显示可用的 Scheme 列表

3. **具体位置**
   - 在运行按钮（▶️）的右侧
   - 在目标设备选择器的左侧
   - 通常显示为：`App > iPhone 15 Pro` 这样的格式

---

## 🎯 如何找到 Scheme 菜单

### 方法 1：通过工具栏

1. 查看 Xcode 窗口顶部
2. 找到运行按钮（▶️，绿色）
3. 运行按钮右侧就是 Scheme 下拉菜单
4. 应该显示类似 "App" 的文字

### 方法 2：通过菜单栏

1. 点击 Xcode 顶部菜单栏的 **"Product"** 菜单
2. 选择 **"Scheme"** 子菜单
3. 选择 **"Edit Scheme..."**

### 方法 3：使用快捷键

直接按快捷键：**`⌘ + <`**（Command + 小于号）

这会直接打开 Scheme 编辑窗口。

---

## 📸 视觉参考

```
┌─────────────────────────────────────────────────────────┐
│  Xcode 顶部工具栏                                        │
├─────────────────────────────────────────────────────────┤
│  ⏹️  ▶️  [App ▼]  [iPhone 15 Pro ▼]  ...              │
│       ↑      ↑                                           │
│    运行    Scheme                                        │
│    按钮   下拉菜单                                       │
└─────────────────────────────────────────────────────────┘
```

---

## 🔧 如果找不到 Scheme 菜单

### 可能的原因

1. **工具栏被隐藏**
   - 解决方法：`View` → `Show Toolbar`（或按 `⌘ + Option + T`）

2. **窗口太小**
   - 解决方法：调整 Xcode 窗口大小，确保工具栏可见

3. **使用快捷键（最简单）**
   - 直接按 `⌘ + <`（Command + 小于号）
   - 这会直接打开 Scheme 编辑窗口

---

## ✅ 推荐方法：使用快捷键

**最简单的方法**：

1. 确保 Xcode 窗口处于活动状态（点击 Xcode 窗口）
2. 按快捷键：**`⌘ + <`**（Command + 小于号）
3. 这会直接打开 "Edit Scheme" 窗口

---

## 📝 打开 Edit Scheme 窗口后的步骤

1. **左侧选择 "Run"**
   - 在左侧列表中，点击 "Run"（应该默认已选中）

2. **切换到 "Options" 标签**
   - 在顶部标签栏中，点击 "Options" 标签

3. **找到 "StoreKit Configuration"**
   - 在 "Options" 标签页中，向下滚动
   - 找到 "StoreKit Configuration" 部分
   - 应该有一个下拉菜单

4. **选择 "Products.storekit"**
   - 点击下拉菜单
   - 选择 "Products.storekit"

5. **保存**
   - 点击窗口底部的 "Close" 按钮

---

## 🎬 完整操作流程（使用快捷键）

1. 在 Xcode 中，按 **`⌘ + <`**
2. 在左侧选择 **"Run"**
3. 点击顶部 **"Options"** 标签
4. 找到 **"StoreKit Configuration"** 下拉菜单
5. 选择 **"Products.storekit"**
6. 点击 **"Close"** 保存

---

## 💡 提示

- 如果快捷键不起作用，确保 Xcode 窗口处于活动状态
- 如果仍然找不到，可以尝试通过菜单栏：`Product` → `Scheme` → `Edit Scheme...`
