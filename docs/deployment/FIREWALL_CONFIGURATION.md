## 配置说明

### 当前配置的含义

- **来源：全部IPv4地址**：允许所有IP地址访问8000端口
  - ✅ **优点**：方便测试，任何地方都可以访问
  - ⚠️ **注意**：安全性较低，任何人都可以访问你的服务

### 生产环境建议（可选）

如果希望提高安全性，可以限制来源IP：

**选项1：限制为特定IP**
- **来源**：`你的公网IP/32`
- **示例**：`123.456.789.0/32`（替换为你的实际IP）

**选项2：限制为IP段**
- **来源**：`你的IP段/24`
- **示例**：`123.456.789.0/24`

**如何查看你的公网IP**：
```bash
# 在本地机器上执行
curl ifconfig.me
```
