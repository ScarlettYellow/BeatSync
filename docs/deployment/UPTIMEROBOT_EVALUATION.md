## UptimeRobot是什么？

**UptimeRobot**：
- 免费的网站/服务监控服务
- 定期检查服务是否可用
- 服务不可用时发送告警通知

**功能**：
- 监控HTTP/HTTPS服务
- 定期ping检查（默认每5分钟）
- 服务不可用时发送邮件/短信通知

## UptimeRobot免费版限制

**免费版限制**：
- 最多50个监控任务
- 检查间隔：5分钟（最短）
- 告警通知：邮件（有限制）
- 历史记录：保留2个月

**对于BeatSync项目**：
- 只需要1个监控任务（健康检查）
- 5分钟检查间隔足够
- 免费版完全够用

### 步骤2：创建或更新监控任务

**如果是新监控任务**：
1. 点击 "Add New Monitor"
2. 选择监控类型：HTTP(s)
3. 填写信息：
   - **Friendly Name**：BeatSync Backend
   - **URL**：`https://124.221.58.149/api/health`
   - **Monitoring Interval**：5 minutes
   - **Alert Contacts**：选择您的邮箱

**如果是更新旧监控任务**：
1. 找到旧的监控任务（如果是监控Render的）
2. 点击 "Edit"
3. 更新URL为：`https://124.221.58.149/api/health`
4. 保存
